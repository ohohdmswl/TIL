### ✔ **contains()**
#JAVA_M/contains
- 참고
	- https://mine-it-record.tistory.com/137

- boolean contains(charSequence)
- 대상 문자열에 특정 문자열 포함 여부 확인
- 대소문자 구분
- 공백도 체크

```java
String str = "hi hello";
System.out.println( str.contains("hi") );  // true
System.out.println( str.contains(" hi") );  // false
```
